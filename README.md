# 🔥 Calories Burnt Analysis and Prediction

A Machine Learning project that predicts the number of calories burned during physical activity using user attributes such as age, gender, height, weight, duration, heart rate, and body temperature.

The project was developed using **Google Colab** and trained with the **XGBoost Regressor** algorithm for accurate predictions.

---

## 📌 Project Overview

The goal of this project is to analyze fitness data and build a regression model that can estimate the calories burned by an individual based on physical and physiological features.

---

## 🚀 Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Selection
- Model Training using XGBoost Regressor
- Model Evaluation
- Predict Calories Burned

---

## 📂 Dataset Features

The dataset contains the following features:

- Gender
- Age
- Height
- Weight
- Duration
- Heart Rate
- Body Temperature
- Calories (Target Variable)
